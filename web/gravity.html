Gravity Simulator

This slightly janky gravity simulator is a play with 3D rendering with raylib, using basic physics equations to show more complex pheomenon, like orbiting bodies and the formation of stars from space dust. It was buit with C and them compiled to Web Assemby by emscripten.

Check out the source code here: https://github.com/ColouredCat/Gravity_Sim

Try raylib for yourself and support the creator raysan here: https://www.raylib.com/
